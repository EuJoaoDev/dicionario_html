Dicionário HTML

**Dicionário baseando-se nas tags que inicialmente eu acho mais interessantes.

  • Tags "Html" e "Body"

    Html é A tag raiz de um documento HTML que encapsula todo o conteúdo da página já o body, Contém todo o conteúdo visível da página web, como textos, imagens e outros elementos.

  • Tags "h1" à "h6"

    Usadas para definir cabeçalhos, com h1 sendo o mais importante e h6 o menos importante. Eles ajudam a estruturar e organizar o conteúdo da página.

  • Tag "p"

    Define um parágrafo de texto. É usada para agrupar blocos de texto e separar diferentes seções de conteúdo.

  • Tag "a"

    Cria um hyperlink. O atributo href especifica o URL de destino. Quando clicado, ele redireciona o usuário para a URL indicada.

  • Tag "img"

    Insere uma imagem na página. O atributo src define o caminho da imagem e alt fornece um texto alternativo que descreve a imagem.

  • Tag "li" e "ul"

    ul, efine uma lista não ordenada (com marcadores) e li Define um item dentro de uma lista

  • Tag "div"

    Define uma divisão ou seção no documento. É usada para agrupar elementos e aplicar estilos ou scripts a esses grupos. Útil para layout e design.

  • Tag "span"

    Usada para aplicar estilos a partes específicas do texto sem criar uma nova linha ou bloco. Ideal para estilização inline.

  • Tag "strong"

    Indica que o texto deve ser exibido com ênfase forte, geralmente renderizado em negrito pelos navegadores.
